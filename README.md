- `src/db/__main__.py` – запускает приложение.
- `src/db/tui.py` – содержит все меню, ввод/вывод, взаимодействие с пользователем.
- `src/db/backend/memory.py` – реализует хранение таблиц в словарях Python, управление таблицами и операции создания, чтения, обновления, удаления записей.

## Функциональность

Проект поддерживает:

- **Создание и удаление нескольких таблиц** (одна таблица по умолчанию – `cars`).
- **Полный набор CRUD операций** для любой таблицы:
  - **Create** – добавление новой записи.
  - **Read** – чтение записей с фильтрацией по одному или нескольким полям.
  - **Update** – обновление записей, соответствующих фильтру.
  - **Delete** – удаление записей, соответствующих фильтру (с подтверждением).
- **Фильтрация** при поиске, обновлении и удалении по любым полям (brand, price, color). Пустые поля игнорируются.
- **Обработка ошибок**:
  - Некорректный ввод (не число в поле price, пустые строки).
  - Обращение к несуществующей таблице или записи.
  - Попытка создать таблицу с уже существующим именем.

- **Консольный интерфейс** с пошаговыми меню и выводом записей в виде таблицы.

### Пример таблицы «cars» (автомобили)

| Поле    | Тип    | Описание          |
|---------|--------|-------------------|
| id      | int    | уникальный номер (генерируется автоматически) |
| brand   | str    | марка автомобиля   |
| price   | int    | цена в руб.        |
| color   | str    | цвет               |
